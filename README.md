#  Data Career Survey Dashboard (Power BI)

<img src="images/overview.png" width="900">

##  Project Overview
This project presents an interactive **Power BI Dashboard** that analyzes survey responses from data professionals worldwide.

The dashboard transforms raw survey data into meaningful insights about **salaries, career paths, skills, job satisfaction, and demographics** using data cleaning, modeling, DAX measures, and interactive visualizations.

It demonstrates a complete **end-to-end Data Analytics workflow**, making it a strong portfolio project for **Data Analyst / BI roles**.

---

##  Dataset Information
- Domain: Data Careers & Job Market  
- Records: 576 survey responses  
- Columns: 24 features  
- Type: Survey dataset  

### Key Columns
- Job Title  
- Career Switch into Data  
- Current Yearly Salary (USD)  
- Industry  
- Favorite Programming Language  
- Satisfaction Scores  
- Difficulty Breaking into Data  
- Gender  
- Age  
- Country  
- Education  
- Ethnicity  
- Time Taken
